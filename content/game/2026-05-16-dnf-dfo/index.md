# 在Ubuntu运行DNF美服版DFO

![](index.assets/361d20c4c69dfda0210ac8f0c39550e4b26ff3bbf4d62e52451a4011cb408954.png)

DNF是个很经典的游戏，但是马服游戏，不充钱玩个屁。如果想要玩的爽一些，可以考虑玩美服。

![](index.assets/a5853cfbe8454acfc1e3f0c21ac97119eb671f01bb7e1d9885da540971745ccd.png)

我的台式机是ubuntu24.04配合KDE桌面

## steam 转译配置

从 https://www.dfoneople.com/ 下载 DFO_Install.exe 后，使用steam 运行DFO_Install.exe 就会自动下载DFO游戏


![](index.assets/e94f35b5de06ffbeaa976f90a1beb79a676c9db9ba28341ced5045cf4ea7cb98.png)

![](index.assets/b689b2f4888324de5743d6c331e8dc38eedf2831255affed15f8878876058b19.png)

![](index.assets/e7350391b8fe042ba27f7379b9147229f5f7a730481b040189f65c8fd3b0a145.png)


![](index.assets/5c54c7bb5dd4f21f2c253b4763da887e4d28822f567a0620279a61e394e2a240.png)

下载完成后，把启动器的目录改成 `"/home/zhaoolee/.local/share/Steam/steamapps/compatdata/3541847699/pfx/drive_c/Neople/DFO/NeopleLauncher.exe"` 开始的位置改成 `"/home/zhaoolee/.local/share/Steam/steamapps/compatdata/3541847699/pfx/drive_c/Neople/DFO"` （把路径的`zhaoolee`换成自己的用户名，其中`3541847699` 是steam分配的序列号，可能每个机器不太一样）


![](index.assets/24936ba2325794ee660ac434d98a7bc3c1a19bd4fa3c8111758479fdaa3dc4f3.png)


通过steam的民间版Proton-GE，可以运行DFO, Proton-GE 可以让AI代劳配置


![](index.assets/386f422753819e44840cdafd02c4350478ca7fc05332e3096a1465766f8cf34e.png)


![](index.assets/5340986fdbb937822d1e184f6b3d57e4577b032414852da0a98fe4d3a01d5bc7.png)
